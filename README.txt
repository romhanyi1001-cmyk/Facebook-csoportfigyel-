Szálláskör figyelő – magyar nyelvű verzió

Ez a projekt egy Facebook csoport figyelő rendszer alapja.
Kulcsszavak alapján találatokat keres, válaszjavaslatot generál,
és Telegram értesítést küld.

Fontos fájlok:
- beallitasok.json → itt tudod beállítani a kulcsszavakat és mintaszövegeket
- titkok.env → ide írod be az API kulcsokat
- figyelo.py → a fő program
